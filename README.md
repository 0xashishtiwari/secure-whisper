<p id="description">An anonymous messaging platform built with Next.js and TypeScript that allows users to receive anonymous messages via a shareable profile link. The platform includes secure authentication using NextAuth email verification powered by Resend and a full sign-up/sign-in flow. Users can view messages sent to them anonymously while their identity remains protected.</p>
